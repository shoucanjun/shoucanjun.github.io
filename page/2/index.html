以前我用macbook的时候，身份是又高贵，又风流，又靓仔啊，同事们都很羡慕，并投以尊敬目光。

如今弃暗投明（穷逼）的我，换了个“X想 windows pro”后，身份地位瞬间全无，还会被嘲讽。

同事们各个拿着新macbook，时不时的用酒精棉片擦拭着心爱的电脑，时刻保持着电脑和电脑屏幕的干净，完全不允许有一点瑕疵出现在电脑屏幕上，这种做法以至于显示他们的高贵儒雅。（以前的我也是这样的）

如果这时你去用手去触碰他们的电脑屏幕留下痕迹的话会被“问候全家身体健康”。（以前的我也是这样的）

而如今如果我拿着酒精棉片去擦拭自己爱心的“X想 windows pro”电脑时，竟然被说到，“你这几千块的辣鸡就不用擦了，浪费酒精棉片”，还被用手指搓了搓屏幕留下污痕而不敢说话，此时的我或许已经到达了人生的低谷。
